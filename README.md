## Setup app
Install the requirements: 
```
pip install -r requirements.txt
```
## Create .env file 
Replace `YOUR_API_TOKEN` with your actual [replicate](https://replicate.com/) api token
```
REPLICATE_API_TOKEN=YOUR_API_TOKEN
```

## Run 
```
gradio script.py
```
