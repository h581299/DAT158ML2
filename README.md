## Setup app
Install the requirements: 
```
pip install -r requirements.txt
```
## Create .env file 
Replace `YOUR_API_TOKEN` with your actual replicate api token `(https://replicate.com/)`
```
REPLICATE_API_TOKEN=YOUR_API_TOKEN
```

## Run 
```
gradio script.py
```
